# Term Project Backend

Steps to setting up this Django project

1) Copy the file .env.example and name the copy .env. On the terminal you can do `cp .env.example .env`

2) Go to this website to generate a Django Secret Key: https://djecrety.ir/

3) Paste the generated secret key in .env

Then, run your project! You'll need to create an app and configure your URLs. See the video from the lecture!
